# Python TCP Port Scanner

## Description

A beginner-friendly TCP port scanner built using Python's built-in `socket` module.

## Features

* Scan a target IP address and TCP port.
* Determine whether a port is open or closed.
* Uses configurable timeouts.

## Technologies

* Python
* Socket Programming
* TCP/IP Networking
